Task Manager API (ASP.NET Core Web API)

This is a simple learning project built to practice ASP.NET Core backend development.
The goal of the project is to understand how to create a Web API, work with CRUD operations, authentication, Entity Framework Core and SQL Server.

Features planned / implemented:
User registration and login (JWT)
CRUD for Tasks (Create, Read, Update, Delete)
Task filtering (completed / pending)
SQL Server database
Entity Framework Core (code-first)
Clean architecture folder structure
Repository & service layers
Basic validation & error handling

Tech stack:
C#
ASP.NET Core Web API
Entity Framework Core
SQL Server
LINQ
JWT authentication

This project is still in progress, as I continue learning .NET and improving my backend skills.
